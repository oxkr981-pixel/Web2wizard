# Web2EXE Wizard

Convert any website into a Windows desktop application with WebView2.

## Features

* Website → Desktop App conversion
* Custom App Name support
* WebView2-based applications
* Automatic Installer Generation (Inno Setup)
* Version Management
* GitHub Update System
* Progress Tracking
* URL Validation
* Custom Icons (planned)

## Requirements

* Windows 10/11
* .NET 8 Runtime
* Microsoft Edge WebView2 Runtime

## How to Use

1. Launch Web2EXE Wizard
2. Enter an App Name
3. Enter a Website URL
4. Click Generate
5. Wait for build completion
6. Find your installer in the output folder

## Example

App Name:

YouTube Desktop

Website URL:

https://www.youtube.com

Output:

YouTube Desktop.exe

YouTube Desktop Setup.exe
