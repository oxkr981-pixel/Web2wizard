# Web2EXE Wizard

Convert any website into a Windows desktop application.

## Features

- Convert websites into native Windows apps
- Automatic icon download from website favicon
- WebView2-based desktop applications
- One-click installer generation
- Version management
- Auto-update manifest generation
- Desktop shortcut creation
- Start Menu integration
- Custom application names

## Requirements

- Windows 10/11
- .NET 8 Runtime
- Microsoft Edge WebView2 Runtime
- Inno Setup (for installer generation)

Before generating installers, install:

- Microsoft Edge WebView2 Runtime
- Inno Setup 6

Inno Setup:
https://jrsoftware.org/isinfo.php

WebView2 Runtime:
https://developer.microsoft.com/microsoft-edge/webview2/

## How to Use

1. Enter an application name.
2. Enter a website URL.
3. Wait for the icon preview to load.
4. Click Generate.
5. Find the installer in:

Template/GeneratedApp/installer/Output
